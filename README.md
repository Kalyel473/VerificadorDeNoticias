# LinkSeguro

> Verificador de URLs - Projeto A3 · Direito Digital · UNP 2026.1

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
![License](https://img.shields.io/badge/License-MIT-green?style=flat)

O **LinkSeguro** permite que qualquer pessoa verifique se uma URL é segura antes de acessá-la, ajudando a combater golpes e desinformação online. O projeto alia segurança da informação a fundamentos jurídicos do Direito Digital, como a LGPD e o Marco Civil da Internet.

---

## Funcionalidades

- Análise de URLs em tempo real
- Identificação de links suspeitos e phishing
- Interface simples, sem necessidade de cadastro
- Aplicação 100% client-side (sem coleta de dados pessoais)

---
## Como Executar

**1. Clone o repositório**

```bash
git clone https://github.com/Kalyel473/VerificadorDeNoticias.git
```

**2. Abra o arquivo no navegador**

Abra o arquivo `index.html` diretamente no navegador. Nenhuma dependência extra é necessária.
