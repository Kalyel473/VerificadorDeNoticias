# LinkSeguro
### Projeto A3 - Direito Digital (UNP)

O LinkSeguro é uma aplicação web desenvolvida como projeto acadêmico (A3) para a disciplina de Direito Digital da UNP. A plataforma permite que qualquer pessoa verifique se um link/URL é seguro antes de acessá-lo, ajudando a combater a desinformação.

O projeto une conhecimentos de segurança da informação com os fundamentos jurídicos do Direito Digital, como a Lei Geral de Proteção de Dados (LGPD) e o Marco Civil da Internet.

---

## Tecnologias Utilizadas

- HTML5, CSS3 e JavaScript

## Como Executar

1. Clone o repositório: `git clone https://https://github.com/Kalyel473/VerificadorDeNoticias.git`
2. Abra o arquivo `index.html` no navegador
